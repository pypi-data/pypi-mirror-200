## Function `subtype(t, name=None)`

Construct a `TypeVar` for subtypes of the type `t`.

Parameters:
* `t`: the type which bounds the `TypeVar`
* `name`: optional name for the `TypeVar`,
  default `t.__name__ + 'SubType'`
